  $ git checkout v1.0-stable

OPTION 2 - Download Release Package:
  Visit: https://github.com/ttshava/edupro-sms/releases/tag/v1.0-stable
  Download: Source code ZIP or TAR.GZ

OPTION 3 - Use Docker (Simplest):
  1. Install Docker Desktop: https://www.docker.com/products/docker-desktop
  2. Clone: git clone https://github.com/ttshava/edupro-sms.git
  3. Checkout: git checkout v1.0-stable
  4. Start: cd frappe_docker && docker compose up -d
  5. Access: http://localhost:8080
